## Overview
This is a deployed personal GenAI project that uses Large Language Models (LLMs)
and prompt engineering to analyze product reviews and generate ratings and
structured insights. The application is built using Streamlit and provides
separate dashboards for users and administrators.

## Features
- AI-powered product review analysis
- LLM-based rating and insight generation
- Prompt engineering for structured outputs
- User-facing Streamlit dashboard
- Admin dashboard for monitoring and analysis
- Fully deployed and publicly accessible app

## Tech Stack
- Python
- Streamlit
- Large Language Models (LLMs)
- Prompt Engineering
- NLP
- Jupyter Notebook

## Project Structure
app/        -> core application logic  
notebooks/ -> prompt experiments and analysis  
data/       -> input datasets  
results/    -> generated outputs
